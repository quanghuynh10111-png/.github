## Đồng bộ hoá GitHub và VS Code

Bạn có thể đồng bộ mã nguồn giữa GitHub và VS Code theo 5 bước nhanh:

1. **Cài Git** và đăng nhập GitHub trên trình duyệt.
2. Trong VS Code, mở **Command Palette** (`Ctrl+Shift+P`) → chọn `Git: Clone` để clone repo từ GitHub.
3. Làm việc trong nhánh của bạn, sau đó vào tab **Source Control** để `Stage` và `Commit`.
4. Bấm **Publish Branch** hoặc **Sync Changes** để đẩy (push/pull) thay đổi với GitHub.
5. Đăng nhập extension **GitHub Pull Requests and Issues** để tạo PR trực tiếp trong VS Code.

Mẹo: bật `Settings Sync` trong VS Code để đồng bộ cài đặt, extension và phím tắt giữa nhiều máy.
